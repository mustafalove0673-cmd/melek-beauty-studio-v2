## SİSTEM NE YAPAR?

NOVA AI, kullanıcı "tema [konu]" dediğinde otomatik olarak Awwwards-seviye premium website üretir. Tek bir komutla:

1. **Konuyu analiz eder** → Sektör template'ine eşleştirir
2. **Renk paletini seçer** → 10 doğru paletten en uygun olanı
3. **3D konsept belirler** → Three.js + R3F ile interaktif 3D sahne
4. **Web'de araştırır** → Trendleri, rakipleri, teknikleri inceler
5. **Tüm section'ları kodlar** → Navbar, Hero3D, Hakkında, Hizmetler, Galeri, İletişim, Footer
6. **AI görseller üretir** → Her bölüm için 8K kalitede görseller
7. **%100 doğruluk testi** → ESLint, compile, responsive, animasyon kontrolü
8. **Kullanıcıya sunar** → Tek seferde, hatasız, Awwwards kalitesinde

**Kullanılan Teknolojiler:** 19 kurulu npm paketi (Three.js, R3F, GSAP, Framer Motion, Lenis, shadcn/ui, Tailwind CSS 4, Next.js 16, Prisma vb.)
