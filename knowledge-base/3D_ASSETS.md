## 🔧 THREE.JS MODEL YÜKLEME

### glTF/GLB Yükleme

```tsx
'use client'
import { useLoader } from '@react-three/fiber'
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader'
import { Suspense } from 'react'

function Model({ url }: { url: string }) {
  const { scene } = useLoader(GLTFLoader, url)
  
  return (
    <primitive
      object={scene}
      scale={1.5}
      position={[0, -1, 0]}
    />
  )
}

export function Scene() {
  return (
    <Suspense fallback={<mesh><boxGeometry /><meshStandardMaterial color="hotpink" /></mesh>}>
      <Model url="/models/villa.glb" />
      <ambientLight intensity={0.5} />
      <directionalLight position={[5, 5, 5]} intensity={1} />
    </Suspense>
  )
}
```

### pmndrs/gltfjsx ile Otomatik Component

```bash
# CLI ile JSX component oluştur
npx @pmndrs/gltfjsx ./model.glb

# Çıktı:
// Auto-generated by: https://github.com/pmndrs/gltfjsx
export default function Model(props) {
  return (
    <group {...props} dispose={null}>
      <mesh geometry={nodes.Cube.geometry} material={materials.Material} />
    </group>
  )
}
```

## 📐 MODEL OPTİMİZASYONU

### Draco Sıkıştırma

```bash
# glTF → Draco sıkıştırılmış glTF
npx gltf-pipeline -i model.gltf -o model.draco.gltf --draco.compressionLevel 7
```

### Three.js Optimizasyon Kuralları

```
1. ✅ .glb kullan (.gltf yerine)
2. ✅ Draco sıkıştırma (50-80% küçülme)
3. ✅ Texture boyutu: max 1024x1024
4. ✅ Polygon sayısı: Desktop 50K, Mobile 10K
5. ✅ useFrame yerine useMemo
6. ✅ dispose() çağır (bellek sızıntısı önle)
7. ❌ 100K+ polygon model mobilde
8. ❌ 2048x2048+ texture web'de
9. ❌ dispose() unutma
```

## 📊 MODEL KALİTE KONTROL LİSTESİ

```
□ Format: .glb (binary, tek dosya)
□ Boyut: < 5MB (ideal), < 10MB (maksimum)
□ Polygon: < 50K (desktop), < 10K (mobile)
□ Texture: < 1024x1024 (web)
□ Material: PBR (metalness + roughness)
□ Animasyon: 30fps, skeleton-based
□ Scale: 1 birim = 1 metre
□ Origin: Modelin ortasında
□ Up axis: Y ekseni (Three.js standard)
□ Normals: Dışarı bakan (outward)
□ Center: (0, 0, 0) noktasında
```
